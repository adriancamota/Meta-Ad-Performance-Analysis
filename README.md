This project is a Power BI dashboard that analyzes paid advertising performance across Facebook and Instagram. It provides insights into reach, engagement, conversions, and budget utilization to help marketing teams evaluate campaign effectiveness and optimize ad strategies.

The dashboard tracks key KPIs such as Impressions, Clicks, Engagements, CTR, Conversion Rate, and Purchases, and breaks performance down by gender, age group, country, ad type, and time. Interactive visualizations highlight high-performing audiences, top ad formats, and optimal posting times.

Overall, the analysis reveals strong engagement and visibility but identifies opportunities to improve conversion efficiency, enabling data-driven decisions for better ROI.
